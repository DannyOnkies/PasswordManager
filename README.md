# 🔐 Password Manager (Python + Tkinter)

Un semplice e leggero **Password Manager** scritto in Python, con interfaccia grafica Tkinter.  
Permette di gestire servizi, username, password e note in modo rapido, con salvataggio automatico su file JSON.

---

## ✨ Funzionalità principali

- Visualizzazione degli account tramite Listbox
- Aggiunta, modifica e rimozione degli account
- Pulizia rapida dei campi (Pulisci Entry)
- Generatore di password con scelta della lunghezza
- Copia password negli appunti
- Ordinamento automatico A→Z dei servizi
- Salvataggio automatico su file JSON
- Interfaccia semplice e immediata

## 🔐 Sicurezza
Le password vengono salvate in chiaro nel file JSON.
Per maggiore sicurezza si consiglia di:

proteggere la cartella con strumenti del sistema operativo

evitare sincronizzazioni cloud non cifrate

valutare in futuro una funzione di crittografia
